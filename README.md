### 使用指南
`:q` 回车退出程序\
`:o[pen]+空格+filename` 打开文件\
如果filename输错直接按回车，再重新从`:o[pen]`开始输入，`:w` 同理保存\
文件名输错下方会显示`Error File`\
`h`, `j`, `k`, `l` 分别为光标左下上右移动\
`/` 搜索文件\
`u` 为撤销，`r` 为重做\
`i` 进入编辑模式\
`x` 删除字母\
下方状态栏依次显示：当前模式 当前指令（仅回显指令）当前文件
